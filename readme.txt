== Description ==
The EMDR Therapist Finder plugin helps users locate EMDR therapists in their area. It features a user-friendly search bar, an interactive map, and a comprehensive list of therapists. The plugin also includes backend functionalities for managing therapist data, custom database tables, and REST API routes for seamless integration.

== Installation ==
1. Upload the `emdr-therapist-finder` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the EMDR Therapist Finder settings page to configure the plugin.

== Usage ==
To display the therapist finder, use the shortcode `[emdr_finder]` in any post or page. This will render the search bar, map, and results list.
